# MAX FIT - Event Management System

## 📌 Project Overview

MAX FIT is a Salesforce-based Event Management System designed to efficiently manage events, organizers, speakers, attendees, and locations. The application automates event registration, validates data, sends email notifications, manages security, and provides Lightning Web Components (LWC) for an enhanced user experience.

## 🚀 Features

### Event Management

* Create and manage events
* Live and recurring event support
* In-person and virtual event types
* Automatic remaining seat calculation
* Event status tracking

### Organizer Management

* Store organizer details
* Associate organizers with events
* Contact and address management

### Attendee Management

* Register attendees for events
* Duplicate prevention
* Confirmation email automation
* Event history tracking

### Speaker Management

* Speaker profile management
* One-event-at-a-time booking validation
* Speaker-event association

### Location Management

* Store complete address information
* Address verification support
* Location lookup functionality
